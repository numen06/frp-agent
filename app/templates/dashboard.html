💡 通过 curl 命令行使用 API

基本用法：
curl -u admin:your_password -X POST \
  -H "Content-Type: text/plain" \
  --data-binary "@frpc.ini" \
  http://localhost:8000/api/frpc/convert/ini-to-toml/direct \
  -o frpc.toml
参数说明：
  • -u admin:your_password - 你的登录用户名和密码
  • --data-binary "@frpc.ini" - 要转换的 INI 文件路径（本地文件）
  • -o frpc.toml - 输出的 TOML 文件名
批量转换示例（Linux/Mac）：
for file in *.ini; do
  curl -u admin:your_password -X POST \
    -H "Content-Type: text/plain" \
    --data-binary "@$file" \
    http://localhost:8000/api/frpc/convert/ini-to-toml/direct \
    -o "${file%.ini}.toml"
  echo "已转换: $file -> ${file%.ini}.toml"
done
Windows PowerShell 示例：
$username = "admin"
$password = "your_password"
$base64 = [Convert]::ToBase64String([Text.Encoding]::ASCII.GetBytes("${username}:${password}"))
$headers = @{
    "Authorization" = "Basic $base64"
    "Content-Type" = "text/plain"
}
$body = Get-Content "frpc.ini" -Raw
Invoke-RestMethod -Uri "http://localhost:8000/api/frpc/convert/ini-to-toml/direct" `
    -Method POST -Headers $headers -Body $body -OutFile "frpc.toml"
