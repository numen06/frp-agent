# frp-agent 前端

基于 Vue 3 + Vite + Tabler UI 的前端应用。

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 构建

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 环境变量

- `.env.development` - 开发环境配置
- `.env.production` - 生产环境配置

## 技术栈

- Vue 3 (Composition API)
- Vite
- Vue Router
- Pinia
- Tabler UI (@tabler/core)
- Tabler Icons (@tabler/icons-vue)
- Axios
